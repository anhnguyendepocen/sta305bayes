Programmer un algorithme d’échantillonnage selon la loi Exponentielle de paramètre \(\lambda\) grâce à la méthode de la fonction inverse et la fonction R runif(). Comparer la distribution de l’échantillon obtenu la densité de probabilité de l’échantillon obtenu avec cet algorithme avec la distribution théorique R dexp().
Explorer différentes valeurs pour le paramètre \(\lambda\) (par exemple 1, 10, 0.78, …) pour vérifier le bon comportement de l’algorithme.

generer_exp <- function(n, lambda) {
    u <- runif(n)
    x <- -1/lambda * log(1 - u)
    return(x)
}

n_ech <- 1000
mon_ech <- generer_exp(n = n_ech, lambda = 10)
hist(mon_ech, probability = TRUE, n = 25)
curve(dexp(x, rate = 10), from = 0, to = max(mon_ech), col = "red", lty = 2, add = TRUE)
legend("topright", c("Fonction inverse", "R dexp()"), lty = c(1, 2), col = c("black", 
    "red"))
